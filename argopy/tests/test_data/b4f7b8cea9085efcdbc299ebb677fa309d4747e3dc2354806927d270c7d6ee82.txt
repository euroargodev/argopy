AB This document is the BGC-Argo quality control manual for radiometry observations. This first version of the document is to apply an initial QC to all radiometry values. Except for data failing the following tests, all the other values will be flagged as probably good data (‘2’). Probably good data need to be validated (i.e., flagged as good “1”) in delay mode.
